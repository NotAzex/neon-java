Every placeholder added by Neon to PlaceholderAPI!

Alive Placeholder: %alive_players%, %alive_size%
 | %alive_players%: Returns alive players.
 | %alive_size%: Returns size of alive players.

Break Placeholder: %breaking_status%
 | %breaking_status%: Returns true/false depending on if breaking is enabled.

Build Placeholder: %building_status%
 | %building_status%: Returns true/false depending on if building is enabled.

Dead Placeholder: %dead_players%, %dead_size%
 | %dead_players%: Returns dead players.
 | %dead_size%: Returns size of dead players.

Fall Damage Placeholder: %falldamage_status%
 | %falldamage_status%: Returns true/false depending on if fall damage is enabled.

Flow Placeholder: %flowing_status%
 | %flowing_status%: Returns true/false

Mutechat Placeholder: %mutechat_status%
 | %mutechat_status%: Returns true/false depending on if chat is muted

PvP Placeholder: %pvp_status%
 | %pvp_status%: Returns true/false depending on if PvP is enabled.

Revival Placeholder: %revival_status%, %revival_number%
 | %revival_status%: Returns true/false depending on if revival is happening.
 | %revival_number%: Returns the answer of the revival. (works, but still WIP)

Revive Recent Placeholder: %reviverecent_players%, %reviverecent_size%
 | %reviverecent_players%: Returns recently dead players.
 | %reviverecent_size%: Returns the size of recently dead players.

Spawn Placeholder: %spawn_location%, %spawn_status%
 | %spawn_location%: Returns the location of the spawn if it's set.
 | %spawn_status%: Returns true/false depending on if the spawn is set.

Timer Placeholder: %timer_formatted%, %timer_raw%, %timer_status%
 | %timer_formatted%: Returns a string with the timer. e.g: (5 minutes, 34 seconds)
 | %timer_raw%: Returns the timer but only as in seconds. e.g: if timer is 5 minutes & 34 seconds, it will return 334
 | %timer_status%: Returns true/false depending on if there's a timer set.

Tokens Placeholder: %tokens_playernamehere%
 | %tokens_playernamehere%: Returns the tokens of the player in the parameters.
